Powerful Capabilities

Built for Speed and Security

Designed with developer experience in mind, supporting all modern token standards and key formats.

🔒

JWE Decryption & Parsing

Automatically detects 5-part JWE compact serialization. Displays protected headers and decrypts payload when provided a key.

🔑

Universal Key Format Support

Supports RSA, ECDSA, Ed25519, and HMAC keys in PEM, DER, JWK, JWKS, X.509 certificates, base64, or raw:<secret> formats.

⏱️

RFC3339 Timestamp Parsing

Converts numeric Unix timestamps (iat, exp, nbf) into human-readable ISO/RFC3339 date strings automatically.

🔄

Recursive Nested Decoding

Handles complex tokens seamlessly: JWT-inside-JWE and JWE-inside-JWE are decoded and pretty-printed recursively.

🎨

TTY-Aware Colored Output

Rich color-coded JSON formatting with automatic TTY detection. Controls C0/C1 characters and targeted bidi controls safely.

📦

Verifiable Cosign Releases

Every release features keyless Cosign signature bundles, Syft SPDX SBOMs, reproducible deb/rpm packages, and multi-arch binaries.

Get Started

Installation & Package Distribution

Install jwtd on macOS, Linux, or Windows with your favorite package manager.

brew install webcodr/tap/jwtd

Official Homebrew formula tap. Updates automatically with each stable release tag.

scoop bucket add webcodr https://github.com/webcodr/scoop-bucket && scoop install jwtd

Official Scoop bucket manifest for Windows amd64 and arm64 architectures.

go install github.com/webcodr/jwtd@latest

Requires Go 1.26 or higher. Compiles directly from source.

sudo dpkg -i jwtd-linux-amd64.deb # Debian / Ubuntu
sudo rpm -i jwtd-linux-amd64.rpm # Fedora / RHEL / openSUSE

Debian and RPM packages place the jwtd binary at /usr/bin/jwtd.

cosign verify-blob --bundle checksums.txt.sigstore.json --certificate-identity-regexp '^https://github.com/webcodr/jwtd/\.github/workflows/release\.yml@' --certificate-oidc-issuer https://token.actions.githubusercontent.com checksums.txt

Verify keyless Sigstore release provenance and payload checksum integrity.
